Lesson #12. Positioning

Static (default)

WARNING! The top, left, right, bottom, z-index properties are not used because the positioning is static.

{ position: relative; }

  • { position: static; top: 30px; z-index: 1; }
  • { position: static; right: 30px; z-index: 2; }
  • { position: static; bottom: 30px; z-index: 1; }
  • { position: static; left: 30px; z-index: 1; }

Relative

  • { position: relative; }
  • { position: relative; top: 10px; left: 10px; }
  • { position: relative; top: -10px; left: -10px; }
  • { position: relative; right: 10px; bottom: 10px; }
  • { position: relative; top: 12px; left: 12px; right: 40px; bottom: 40px;}

  • li { height: auto; }

    li > div { position: relative; left: 20%; bottom: 20%;}

  • li { height: 80px; }

    li > div { position: relative; left: 20%; bottom: 20%;}

  • Text span { position: relative; }

z-index

  • { position: relative; top: 13px; z-index: 1; }

    { position: static; z-index: 2; }

  • { position: relative; top: 13px; z-index: 1; }

    { position: relative; z-index: 1; }

  • { position: relative; top: 15px; z-index: 10000;}

    { position: relative; z-index: 9999;}

  • { position: relative; top: 15px; z-index: 1;}

    { position: relative; z-index: -1;}

  • { position: relative; z-index: 2; }

    { z-index: 5; }

    { z-index: 3; }

    { position: relative; z-index: 4; }

    { z-index: 2; }

    { z-index: 1; }

  • { position: relative; z-index: 2; }

    { z-index: 5; }

    { z-index: 3; }

    { position: static; z-index: 4; }

    { z-index: 2; }

    { z-index: 1; }

Absolute

{ position: absolute; top: -5px; right: -5px; }

{ position: absolute; top: 10%; bottom: 10%; right: 0; }

{ position: absolute; bottom: 30%; right: 100px; }

Parent
{ position: relative; }

Child
{ position: absolute; top: 100%; left: 100%; }

{ position: absolute; left: 20%; right: 30%; bottom: 10%; }

{ position: absolute; right: 100%; }

Fixed

As examples, 2 blocks and a site header

Positioned relative to the browser window!!

{ position: fixed; top: 0; right: 0; z-index: 10006; }

{ position: fixed; bottom: 0; left: 0; }
